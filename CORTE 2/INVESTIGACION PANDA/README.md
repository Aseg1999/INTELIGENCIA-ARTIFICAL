# Temas investigados

## ¿Qué es Pandas?

Pandas es una librería de Python utilizada para trabajar con datos estructurados mediante herramientas que permiten organizar, transformar y analizar información.

Sus estructuras principales son:

- Series
- DataFrame

---

## ¿Para qué sirve?

Pandas permite:

- Leer archivos.
- Organizar datos.
- Realizar operaciones matemáticas.
- Obtener estadísticas.
- Procesar información.

---

## Objetivo principal de la librería

Facilitar el procesamiento y análisis de datos mediante estructuras eficientes.

---

## Principales funciones y características

Funciones estudiadas:

- DataFrame()
- read_csv()
- info()

- head()
- groupby()

Características:

- Fácil implementación.

## Importancia de Pandas en Inteligencia Artificial y análisis de datos

Pandas permite preparar conjuntos de datos antes del entrenamiento de modelos de Inteligencia Artificial.

Sus usos incluyen:

- Limpieza de datos.
